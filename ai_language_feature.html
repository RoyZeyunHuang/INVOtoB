Module 02
Outbound renewal reminders
INVO generates personalised renewal reminder emails in the resident's detected language. The template is designed to increase renewal conversion — clear, warm, and professional. Language is selected automatically based on name analysis and message history.
◆
How language is auto-detected
INVO parses the resident's full name to identify likely language background, then cross-references all prior inbound messages. If the resident has written in Chinese three times and English once, Chinese is selected. The system also detects script direction (RTL for Arabic) and applies it automatically to the outgoing email.
